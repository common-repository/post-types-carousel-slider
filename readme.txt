Post Carousel & Slider is simple way to build slider and carousel.

== Description ==

You can create a post slider and carousel for your WordPress site. All you need to do is just a few clicks and your slider will be ready.

[DEMO](http://nioc.co.in/post-slider/)
[SCREEN RECORDING](https://youtu.be/bg5817yOVOo)

The slider uses auto-generated shortcodes for making it easier to add it to the custom location.

== Features ==
01. You can create multiple post sliders with different options at single page or post
02. Custom post type support
03. Ascending or descending order and Sort retrieved posts
04. Show posts associated with certain categories, taxonomies & tags
05. Show posts based on a keyword search
06. CSS animation [https://daneden.github.io/animate.css/](https://daneden.github.io/animate.css/)
07. Given multiple design option
08. Work with Mousewheel & Touch swipe

== Installation ==

= Admin Installer via search =
1. Visit the Add New plugin screen and search for "Post Carousel & Slider".
2. Click the "Install Now" button.
3. Activate the plugin.
4. Navigate to the "Post Carousel & Slider" Menu.

= Admin Installer via zip =
1. Visit the Add New plugin screen and click the "Upload Plugin" button.
2. Click the "Browse..." button and select zip file from your computer.
3. Click "Install Now" button.
4. Once done uploading, activate Post Carousel & Slider.

= Manual =
1. Upload the Post Carousel & Slider folder to the plugins directory in your WordPress installation.
2. Activate the plugin.
3. Navigate to the "Post Carousel & Slider" Menu.
